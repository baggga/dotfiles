# Development configs
Tmux and vim configuration files.

## Prerequisites
Tmux and vim need to be installed.

## Installing

Clone repository and run install script:
```
git clone https://github.com/art-vasilyev/dotfiles.git && cd dotfiles && ./install.sh
```
